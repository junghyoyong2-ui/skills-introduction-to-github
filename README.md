# 부산대학교병원 법무지원팀 프롬프트 템플릿 뷰어

브라우저에서 바로 실행할 수 있는 **업무용 프롬프트 템플릿 조회 화면**입니다.

## 무엇을 하는 도구인가요?
- 사건 관리/계약/보고/커뮤니케이션 템플릿을 찾고
- 클릭해서 내용을 확인한 뒤
- **복사 버튼으로 바로 붙여넣기**할 수 있게 해줍니다.

## 실행 방법 (가장 쉬운 방법)
1. 이 저장소 폴더에서 `index.html` 파일을 더블클릭합니다.
2. 브라우저가 열리면 바로 사용 가능합니다.

## 실행 방법 (로컬 서버 사용)
```bash
python3 -m http.server 8000
```
그다음 브라우저에서 `http://localhost:8000`으로 접속하세요.

## 사용 방법
1. 검색창 또는 카테고리에서 원하는 템플릿을 찾습니다.
2. 템플릿을 클릭하면 오른쪽에 내용이 표시됩니다.
3. **복사** 버튼을 누른 뒤, ChatGPT 대화창에 붙여넣어 요청합니다.

## 사용 시 주의
- 환자/직원 개인정보는 반드시 비식별화 후 입력하세요.
- 최종 대외 문서는 내부 결재 및 검토 절차를 거쳐 확정하세요.
