# 미로 수학 탐험대 (8살 어린이를 위한 교육용 게임)

브라우저에서 바로 실행할 수 있는 미로형 사칙연산 학습 게임입니다.

## 기능
- 미로 길 위에서 문제를 맞히면 한 칸씩 전진
- 초등 저학년용 덧셈/뺄셈/곱셈 문제
- **받아올림 없는 덧셈**, **받아내림 없는 뺄셈**만 출제
- 정답/오답 시 각각 다른 효과음 재생
- 점수, 현재 위치, 남은 기회 표시

## 실행 방법
1. 이 저장소를 클론합니다.
2. `index.html` 파일을 브라우저에서 엽니다.

간단히 실행하려면 아래 명령어를 사용할 수 있습니다.

```bash
python3 -m http.server 8000
```

그다음 브라우저에서 `http://localhost:8000`으로 접속하세요.
