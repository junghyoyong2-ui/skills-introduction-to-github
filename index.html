🏥 법무지원팀 업무용 프롬프트 템플릿

필요할 때 카테고리를 고르고 템플릿을 클릭하면 바로 복사할 수 있어요. 민감정보는 반드시 비식별화해서 사용하세요.

빠른 실행 가이드

  1. 이 폴더에서 index.html을 더블클릭해서 브라우저로 엽니다.
  2. 또는 터미널에서 python3 -m http.server 8000 실행 후 http://localhost:8000 접속.
  3. 카테고리/검색으로 템플릿 선택 → 복사 버튼 → 채팅창에 붙여넣기.

템플릿 목록

    템플릿을 선택하세요

    왼쪽 목록에서 원하는 템플릿을 누르면, 프롬프트 본문을 확인하고 복사할 수 있습니다. 

    선택된 템플릿이 없습니다.

    복사 버튼을 누르면 클립보드에 저장됩니다.
